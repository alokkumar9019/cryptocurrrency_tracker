# 🚀 CryptoCurrency Tracker

A modern and responsive Cryptocurrency Tracker web application built using React.js that allows users to track real-time cryptocurrency prices, market trends, and detailed coin information.

---

## 🌟 Features

- 📈 Real-time cryptocurrency prices
- 🔍 Search cryptocurrencies by name
- 📊 Market cap, volume, and price change tracking
- 📱 Fully responsive design
- ⚡ Fast and optimized UI
- 🌙 Clean and modern interface
- 🔄 Live API data integration

---

## 🛠️ Tech Stack

- React.js
- JavaScript
- CSS / Tailwind CSS
- REST API
- Axios

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/crypto-tracker.git
```

### Navigate to project directory

```bash
cd crypto-tracker
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```
